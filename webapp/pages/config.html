Telegram Bot

  1. Download Telegram
    2. Download Telegram if you haven't already.
  2. Log in
    3. Log in into Telegram using your phonenumber.
  3. Search for "BotFather"
    4. In the top right corner of the app there should be a search-icon. If you use the web or desktop version of Telegram there should be a seachbar on the top left.
    Now search for "BotFather".
  4. Start "BotFather"
    5. When you now select the BotFather there should appear a "START" button on the bottom of the screen. Now simply click on this button to start the Bot.
    If this button doesn't appear type and send "/start".
  5. Create a new Bot
    6. The BotFather should now have sent you a message with all commands. Select "/newbot" or send it as a message.
  6. Choose a name
    7. The BotFather will ask you to choose a name for your Notification-Bot.
    Send him a message with the name you want. We suggest "CamelBot". ;)
  7. Choose a username
    8. Next you have to choose the Bot's username for Telegram.
    The username has to be unique, so it might take a few attempts to find one which is not taken already.
    Something like "[your name]-camelbot" will do the trick.
  8. Submit the token
    9. Your Bot is now created.
    Copy, paste and save the token, which you got from the Bot, in the input-field below. (You could also type in the token letter by letter if you have too much time)

  9. Open your Bot-chat
    10. In the same message as the token, there should also be a link, which leads you to your own bot.
    The link should look something like this: "t.me/[bot-username]". Klick on that link.
  10. Start your Bot
    11. Start your own Bot the same way you started the BotFather.
  11. Congratulations, you now have your own CamelBot!
    12. Your Bot now sends you a message with all the commands it supports.
    Have fun using CamelBot!
